# Identify requirements for changes to manufacturing/delivery processes

## Overview

Identifying any changes that need to be effectuated in the organization's internal processes for manufacturing, and delivering the newly developed products/services. Determine if any changes need to be made to the production and distribution processes, in light of the new products/services. Begin production process planning. Prepare for factory layout planning. Generate shop-floor instructions and changes to the supply chain. Within the Aerospace and Defense industry, this process takes on particular importance due to the sector's emphasis on precision manufacturing, safety-critical systems, and defense program management. Organizations must ensure alignment with FAA, ITAR, DFARS, AS9100 standards while optimizing for efficiency and effectiveness.
