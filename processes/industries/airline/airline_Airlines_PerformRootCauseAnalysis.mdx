# Perform root cause analysis

## Overview

Using a technique that helps people answer the question of why a problem occurred in the first place. It seeks to identify the origin of a problem using a specific set of steps, with associated tools, to find the primary cause of the problem to determine what happened, why, and how to reduce the likelihood that it will happen again. Within the Airline industry, this process takes on particular importance due to the sector's emphasis on flight operations, passenger experience, and maintenance turnaround. Organizations must ensure alignment with FAA, TSA, DOT, IATA standards while optimizing for efficiency and effectiveness.
