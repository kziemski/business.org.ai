# Manage IT user requests

## Overview

Creating an effective plan and structure to address and resolve requests of IT users. Determine, record, and monitor user requests. Obtain information about the effectiveness and performance of the user request handling process from the IT users through various means. Within the Airline industry, this process takes on particular importance due to the sector's emphasis on flight operations, passenger experience, and maintenance turnaround. Organizations must ensure alignment with FAA, TSA, DOT, IATA standards while optimizing for efficiency and effectiveness.
